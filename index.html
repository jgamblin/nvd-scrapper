Public mirror of the full NVD CVE dataset, refreshed every 3 hours.

Endpoints

  • /nvd.json — full dataset as a JSON array
  • /nvd.jsonl — byte-identical copy (legacy filename)
  • /metadata.json — last run timestamp, CVE count, duration
